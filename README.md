# godis
redis server implemented with go

> 目标：通过go实现一个基础功能的Redis服务，来达到更加深刻理解Redis原理的目的。

## 开发计划

* 实现Redis核心 "骨架" 功能

    - [ ] 实现一个服务端
        + [ ] 协议解析
        + [ ] 网络处理
    - [ ] 实现一个客户端
    - [ ] 实现 `SDS` 基础数据类型的操作

* 完善支持的数据结构类型
    
    - [ ] 实现 `链表` 
    - [ ] 实现 `字典`
    - [ ] 实现 `HASH`
    - [ ] ...

* 进阶功能

    - [ ] 多数据库
    - [ ] 数据持久化
* 优化
    
    - [ ] 参数解析
    - [ ] 服务监控
    - [ ] 服务重启
